Add a :root CSS variable block to frontend/src/index.css immediately after the `@import "tailwindcss"` line and before the `@media print` block.

The exact block to insert:

```css
:root {
  /* Warm accent: emerald-700 range — replaces zinc-900 primary (D-03) */
  --primary: oklch(0.45 0.14 155);            /* ≈ emerald-700 / #2d7a4f */
  --primary-foreground: oklch(0.98 0.01 155); /* near-white for contrast on emerald background */
}
```

Color rationale (Claude's discretion per CONTEXT.md):
- `oklch(0.45 0.14 155)` — L=0.45 (medium-dark), C=0.14 (vivid), H=155 (green hue). Maps to approximately Tailwind emerald-700 (#2d7a4f). Reads well on white backgrounds.
- `oklch(0.98 0.01 155)` — Near-white with a faint green tint. Ensures WCAG AA contrast on the emerald background.

Final file structure must be:

```css
@import "tailwindcss";

:root {
  /* Warm accent: emerald-700 range — replaces zinc-900 primary (D-03) */
  --primary: oklch(0.45 0.14 155);

@media print {
  body { font-size: 12pt; line-height: 1.5; }

  .recipe-display-name { font-size: 18pt; font-weight: 700; }
}

@page { margin: 1cm; }
```

Do NOT change or remove any existing print CSS rules. Do NOT add any other CSS beyond the :root block.
